> 💡 **실시간 프레젠테이션 편집 시스템**  
> CRDT 알고리즘과 하이브리드 통신 구조를 활용한 실시간 프레젠테이션 편집 도구입니다.

---

## 🧭 프로젝트 소개

**Kimbanana**는 다중 사용자가 동시에 슬라이드를 편집할 때 발생하는 데이터 충돌 문제를 해결하고,
슬라이드 단위의 세밀한 버전 관리를 지원하는 **실시간 협업 웹 애플리케이션**입니다.
- 충돌 없는 협업: CRDT(Yjs) 도입으로 동시 편집 시 데이터 정합성 확보
- 안정적인 통신: WebSocket과 REST API를 병행하는 하이브리드 구조 설계
- 정밀한 복구: 전체 문서가 아닌 슬라이드별 독립적 히스토리 관리 및 복원 기능

---

## ⚙️ 설치 및 실행

```bash
pnpm install
pnpm dev
```
